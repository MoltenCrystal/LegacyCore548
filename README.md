## Requirements
+ Platform: Linux, Windows or Mac
+ Processor with SSE2 support
+ Boost 1.78
+ MySQL = 8.0 - 8.4 LTS
+ cmake >= 3.24 (Windows / Linux)
+ OpenSSL= 3.x.x
+ MS Visual Studio = 17 (2022) (Windows only)
